
PDF.js для GitHub Pages
========================

Содержимое этой папки — это собранный pdf.js viewer для использования в GitHub Pages.

Как использовать:
-----------------
1. Загрузите всю папку `pdfjs` в ваш репозиторий GitHub (в корень).
2. Убедитесь, что GitHub Pages активирован (ветка main, папка root).
3. Для предпросмотра PDF используйте ссылки вида:

   https://ВАШ_ЮЗЕР.github.io/pdfjs/web/viewer.html?file=https://ВАШ_ЮЗЕР.github.io/ИМЯ_ФАЙЛА.pdf

Например:
   https://ranepalip.github.io/pdfjs/web/viewer.html?file=https://ranepalip.github.io/sample.pdf
